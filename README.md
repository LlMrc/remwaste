# RemWaste

RemWaste is a web application designed to help users manage and track their waste disposal effectively. The application allows users to log, update, and view waste disposal activities with a focus on environmental sustainability.

## Features

- **User-Friendly Interface**: Easy-to-navigate interface for tracking waste management activities.
- **Responsive Design**: The app is optimized for both desktop and mobile devices.
- **Real-Time Data**: Keep track of your waste disposal records in real time.
- **GitHub Pages Deployment**: View the live version of the project.

## Live Version

You can access the live version of the project by clicking on the link below:

[**Live Demo**](https://llmrc.github.io/remwaste/)

## Technologies Used

- **React**: For building the user interface.
- **Vite**: For fast development and build tools.
- **gh-pages**: For deploying the application to GitHub Pages.

## Setup Instructions

To run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/LlMrc/remwaste.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open the application in your browser at `http://localhost:3000`.

## Deployment

To deploy the project to GitHub Pages:

1. Build the project:
   ```bash
   npm run build
   ```

2. Deploy to GitHub Pages:
   ```bash
   npm run deploy
   ```
